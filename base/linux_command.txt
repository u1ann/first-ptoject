cd name - перемешает нас в какую-то папку (директорию)
ls - просматривает содержимое директории
touch name - создание файла
mkdir name - создает папку(директорию)

ls -a - показывает файлы и папки вместе со скрытыми файлами и папками
ls -l - показывает файлы и папки с доп.информацией
ls -ls - показывает файлы и папки со скрытыми файлами и папками + доп инфирмоация

cd - переходит в домашнюю папку
cd ~ переходит в домашнюю папку
cd . переходит в ту папку в которой вы находитесь
cd .. - переходит в предыщую папку (родительская папка)

rm name - удаление файла name 
rmdir -  удаление пустой директории(папки)
rm -rf -удаляет все: папки с содержимым, файлы, пустые папки

history - выводит историю команд
history какое-то число - выводит историю в каком-то количестве
history | grep слово - выводит все команды, в которых есть указанное слово


pwd - показывает абсолютный путь (начиная с корневой папки) до текущей директории в которой вы находитесь

cat - выводит содержимое файлов
 
nano - текстовый редактор, при помощи которого мы можем записать данные в файла

mv test1.txt test3.txt -  переименовает файл test1.txt в  test3.txt
mv test1 test3 - переименовает папку test1 в test3
mv test1.txt  test1 - перемещает файл test1.txt в папку test1


apt - системный установщик пакетов
snap - установщик пакетов (чаще всего использует для установки программ)

sudo - (superuser do) - выполнение какой-либо команды от имени администратора
